1 Basic commands

1.1 Open/Save/Close

Short cut keyDescription
C-x C-fOpen a file
C-x C-sSave a file
C-x C-cClose the emacs

1.2 Toggle readonly

C-x C-q

1.3 Cut/Copy/Paste

Short cut keyDescription
C-wCut / kill
M-wCopy / Yank
C-yPaste

1.4 Search/Find

Short cut keyDescription
C-sIncremental search forward (isearch-forward)
C-rIncremental search backward (isearch-backward)
C-M-sForward Regular Expression search
C-M-rBackward Regular Expression search
C-s C-wSearch a string under cursor:
You can type Ctrl+w multiple times to expand the word selection

1.5 List Lines Matching a String, for a Single File

M-x list-matching-lines

1.6 Grep and list all

M-x lgrep pattern

1.7 Undo

C-x u

1.8 Panes/Buffers

Short cut keyDescription
C-x oMove to other pane
C-x 1Maximize Pane
C-x bSwitch to Buffer
C-x kKill buffer
C-x C-bList the buffers

1.9 Emacs tutorial

Short cut keyDescription
C-h tDisplay Emacs tutorial
C-h fHelp for fuction
C-h ?Help
